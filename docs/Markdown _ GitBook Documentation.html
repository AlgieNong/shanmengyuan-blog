Markdown

在编辑器中直接编写 Markdown，使用常见语法轻松创建内容

GitBook 的编辑器允许你使用 Markdown 创建格式化内容。

Markdown 是一种以简洁著称的流行标记语法。GitBook 支持它作为一种便于键盘输入的方式来编写富文本和结构化文本。

你可以通过访问以下内容了解更多关于 Markdown 本身的信息 Common Mark.

文本格式化

GitBook 支持所有经典的行内 Markdown 格式：

粘贴 Markdown

当直接将 Markdown 内容粘贴到编辑器中时，重要的是使用 粘贴并匹配样式 选项（通常是 Shift + Cmd + V 在 Mac 上或 Shift + Ctrl + V 在 Windows 上）。

如果你对从其他编辑器或网页复制的内容使用常规粘贴选项，可能会将其作为代码块插入，而不是格式化的文本。

  • 二级标题： ## 二级标题

  • 三级标题： ### 三级标题
